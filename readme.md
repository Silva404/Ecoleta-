## Getting Started

### Prerequisites

To run this project in development mode locally, you'll need to have a few dependencies.
* Node 12.18.0
* Express 4.17.1
* Nunjucks 3.2.1
* Sqlite 4.2.0

Dev dependencie
* Nodemon

OS & Linux: 
```
sudo apt-get install -y nodejs
```
```
npm install express
```
```
npm install sqlite3
```
```
npm install nunjucks
```

```
npm install nodemon
```

### Installing
